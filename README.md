## Giới thiệu

Hệ thống chatbot hỗ trợ tư vấn tuyển sinh, cho phép:

- tra cứu ngành học
- tra cứu học phí
- tra cứu phương thức xét tuyển
- tra cứu tổ hợp môn
- tra cứu điểm chuẩn
- hỏi đáp từ tài liệu tuyển sinh bằng RAG
